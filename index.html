Acerca del Blog Psicoandino

Este blog es un experimento creativo y técnico. Está construido solo con HTML, CSS y JavaScript puro, sin backend, y se puede alojar gratis en GitHub Pages.

Para agregar nuevas entradas solo hay que editar el archivo posts.js y agregar un nuevo objeto al arreglo de posts.
